# To extract widgets

1. Select the widget code in your .dart file.

2. Right-click → Refactor → Extract Widget…
    - OR use the shortcut:
    - Windows/Linux: Ctrl + Alt + W
    - Mac: ⌘ + Option + W

3. Give your new widget a name and confirm.
